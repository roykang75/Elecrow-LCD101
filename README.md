# Elecrow-LCD101

Steps for setting up the touch function for Raspberry Pi:


Step 1: Open terminal and Download the driver on RPI
Run:
git clone https://github.com/Elecrow-keen/Elecrow-LCD101.git


Step 2: Install driver
Run:
cd Elecrow-LCD101
sudo ./Elecrow-LCD101
